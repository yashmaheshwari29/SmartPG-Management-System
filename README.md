# Smart PG Management System

## Overview

A full-stack PG management platform developed using AngularJS, PHP, and MySQL.

## Features

- Tenant Management
- Room Management
- Complaint Management
- Payment Tracking
- Admin Dashboard
- Role-Based Access Control

## Technologies Used

- HTML
- CSS
- AngularJS
- PHP
- MySQL
- XAMPP

## Architecture

3-Tier Architecture:
Frontend → PHP APIs → MySQL Database
